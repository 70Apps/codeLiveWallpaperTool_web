# iPad Screen Size Reference

This page lists common Apple iPad models and their nominal pixel dimensions (portrait orientation). Use these values as a reference when preparing wallpapers or design assets.

## 2026 iPad lineup (reference)

> Common iPad screen sizes and pixel resolutions (portrait).

| Model | Screen Size | Pixels (px) | Notes |
|:---|:---:|:---:|:---|
| iPad Pro 12.9" (6th gen) | 12.9" | 2048 x 2732 | 2732×2048 in landscape |
| iPad Pro 11" (4th gen) | 11" | 1668 x 2388 | 2388×1668 in landscape |
| iPad Air 10.9" | 10.9" | 1640 x 2360 | typical Air resolution |
| iPad 10.2" | 10.2" | 1620 x 2160 | standard non-Pro iPad |
| iPad mini 8.3" | 8.3" | 1488 x 2266 | compact iPad mini |

## Notes and usage

- Pixel values are shown in portrait orientation (width × height). For landscape wallpapers, swap the numbers.
- When generating wallpapers, target the exact device pixel size for best fit, or use center-crop/contain strategies to preserve important content.
- For multi-device outputs, consider exporting assets at the device pixel size and providing higher-resolution versions where desired.
